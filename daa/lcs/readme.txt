Input file: INPUT.TXT (TEXT & PATTERN)
Output file: OUTPUT.TXT

L1: length of TEXT
L2: length of PATTERN

-10=no direction (NO)
-1=left (LT)
0=diagonal (DI)
1=up (UP)

MATRIX: C of L1+1 X L2+1; DIR of L1+1 X L2+1 for direction value

convention taken UP as default direction.

Function: PRINT_LCS to print longest common sub-sequence
